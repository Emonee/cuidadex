# Cuidadex

A simple page for cats and dogs breeds

# Requiriments

| Tech                            | Version |
|---------------------------------|---------|
| [NodeJS](https://nodejs.org/en) | LTS     |

# Install the project

```shell
git clone git@github.com:Emonee/cuidadex.git
cd cuidadex
npm i
```

# Run de project on development mode

```shell
npm run dev
```
